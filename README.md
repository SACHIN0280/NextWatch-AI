# 🎬 NextWatchAI – Movie Recommender System

NextWatchAI is an AI-powered Movie Recommendation System built using **Python**, **Streamlit**, and **Machine Learning**.
The application recommends similar movies based on user selection using content-based filtering techniques.

---

# 🚀 Features

* 🎥 Recommend movies instantly
* 🤖 AI-powered recommendation engine
* ⭐ Clean and modern UI
* 📌 Displays movie posters
* 🔍 Search from thousands of movies
* ⚡ Fast and lightweight Streamlit app

# ⚙️ How It Works

The system uses **Content-Based Filtering** to recommend movies.

1. Movie data is preprocessed
2. Features like genres, keywords, cast, and overview are combined
3. Text vectorization is applied
4. Cosine similarity is calculated
5. Similar movies are recommended based on selected movie
